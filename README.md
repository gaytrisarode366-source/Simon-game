# 🎮 Simon Says Game

A simple memory-based game built using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo
👉 https://gaytrisarode366-source.github.io/Simon-game/

---

## 🧠 How to Play

1. Press any key to start the game  
2. A color will flash on the screen  
3. Click the same color  
4. Sequence increases every level  
5. If you click the wrong button → Game Over ❌
